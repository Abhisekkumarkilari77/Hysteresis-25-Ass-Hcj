# Notes App (Local Storage Only)

Vanilla HTML/CSS/JavaScript notes app that stores everything in Browser Local Storage.

Open `index.html` in your browser to run.

Features:
- Create / Read / Update / Delete notes
- Search by title/content
- Filter by category
- Sort by newest/oldest
- View full note in modal with Edit/Delete/Pin
- Pinned notes show first
- Dark/Light theme (saved)
- Character counter and draft auto-save
- Export all notes to a text file
